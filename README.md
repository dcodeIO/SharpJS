=====================================================================
**SharpJS** is an embeddable, event-driven, node.js-like JavaScript environment for the Mono/.NET platform.

Implemented
-----------
* assert
* buffer
* console
* events
* fs
* path
* process (partly)
* querystring
* string_decoder
* timers
* util (partly)

Not yet implemented
-------------------
* child_process
* cluster
* crypto
* debugger
* dns
* domain
* http
* https
* net
* os
* punycode
* readline
* repl
* stream
* tls/ssl
* tty
* udp
* url
* vm
* zlib
